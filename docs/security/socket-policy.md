# Socket.dev Alert Policy

Block confirmed malware, typosquatting, dependency confusion, compromised
ownership, high/critical vulnerabilities, and unexplained binary payloads.
Review shell/network access, dynamic execution, install scripts, native code,
opaque archives, and AI findings. Monitor verified scientific native code and
informational quality signals.

Suppressions must be exact-version and exact-artifact decisions with a SHA-256,
owner, evidence, expiry, and reassessment trigger. Blanket ignores are not
permitted.
